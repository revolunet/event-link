# Event Link

Generate shareable calendar event links for Google Calendar, Outlook, and iCal.

## Features

- Create calendar events with title, description, date/time, and URL
- Generate links for Google Calendar, Outlook, and iCal (.ics download)
- Share events via URL - recipients see a clean buttons-only view
- Copy links to clipboard with one click
- Works entirely client-side - no server required

## Usage

1. Fill in your event details
2. Click any calendar button to add the event
3. Click **Share Event** to generate a shareable link
4. Recipients can add the event to their calendar with one click

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm test
npm run test:e2e

# Build for production
npm run build
```
